Tetris has a temporary dependency on GLFW. 

## macOS
`brew install glfw` 
 
## Ubuntu 
`sudo apt install libglfw3 libglfw3-dev libfreetype6-dev libssl-dev`

## Arch (and Manjaro)
`sudo pacman -S glfw-x11` 

## Windows 
Copy `thirdparty/glfw/glfw3.dll` to this directory.
